# AI Ops Lab

Лендинг для направления AI-интеграций, AI-агентов, RPA, RAG-систем, n8n-сценариев и CRM/ERP-автоматизации.

## Файлы

- `index.html` — основной сайт
- `robots.txt` — разрешение индексации
- `sitemap.xml` — карта сайта

## Публикация на GitHub Pages

1. Создайте новый репозиторий, например `ai-ops-lab`.
2. Загрузите файлы из архива в корень репозитория.
3. Откройте `Settings → Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохраните.
7. Сайт откроется по адресу вида: `https://USERNAME.github.io/ai-ops-lab/`.
